# Equinozio App — Tema 3 (Rifinitura UI/UX & accessibilità) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Tech Stack:** SwiftUI, SwiftData. Lavorare su `main`.

**Spec:** `docs/superpowers/specs/2026-06-03-equinozio-landing-e-app-design.md` (sez. 4, Tema 3).

**Fuori scope (deciso):** centralizzazione completa delle stringhe (refactor ampio, basso valore per app solo-italiana). La duplicazione del DateFormatter è già un task separato.

## Task 2: Banner introduttivi ri-mostrabili

I banner intro di Riflessione e Storico sono nascosti per sempre dopo "HO CAPITO" (`@AppStorage` flag). Aggiungere un bottone "?" che li ri-mostra.

**Files:** RiflessioneView.swift, StoricoRiflessioniView.swift

- [ ] **Step 1: Bottone "?" nella Riflessione**

In `Equinozio/Features/Riflessione/RiflessioneView.swift`, nell'`intestazione`, dentro l'`HStack`, prima dello `Spacer()` o accanto al bottone STORICO, aggiungi un bottone che riattiva l'intro. Sostituisci la riga `Spacer()` dell'intestazione con:
```swift
            Spacer()
            if introLetta {
                Button {
                    withAnimation { introLetta = false }
                } label: {
                    Image(systemName: "questionmark.circle")
                        .font(.system(size: 16, weight: .light))
                        .foregroundStyle(Color.attenuato)
                }
                .buttonStyle(.plain)
                .accessibilityLabel("Mostra spiegazione della riflessione")
                .padding(.trailing, S.x2)
            }
```

- [ ] **Step 2: Bottone "?" nello Storico**

In `Equinozio/Features/Riflessione/StoricoRiflessioniView.swift`, aggiungi un ToolbarItem accanto al titolo. Nel `.toolbar { ... }`, aggiungi dopo il `ToolbarItem(placement: .principal){...}`:
```swift
                ToolbarItem(placement: .topBarTrailing) {
                    if introLetta {
                        Button {
                            withAnimation { introLetta = false }
                        } label: {
                            Image(systemName: "questionmark.circle")
                                .font(.system(size: 16, weight: .light))
                        }
                        .tint(.attenuato)
                        .accessibilityLabel("Mostra spiegazione dello storico")
                    }
                }
```

- [ ] **Step 3: Build** → BUILD SUCCEEDED.
