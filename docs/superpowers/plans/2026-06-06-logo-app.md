# Logo unificato — App iOS Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** Portare l'emblema dell'app iOS al logo canonico "quattro cerchi a rombo" (passione↑/talento←/missione→/professione↓), senza puntino centrale, e rigenerare l'icona app col nuovo logo.

**Architecture:** Modifica del componente sorgente `QuattroCerchi.swift` (Canvas SwiftUI) — la geometria a rombo si propaga a Mappa/Splash/Esplorazione/Blocco. L'icona app è un PNG 1024 opaco rigenerato da un SVG sorgente del rombo su sfondo salvia.

**Tech Stack:** SwiftUI (Canvas), `rsvg-convert` + `magick` per SVG→PNG.

**Riferimento:** `docs/superpowers/specs/2026-06-06-logo-unificato-design.md`.

**Convenzioni:**
- Build: `xcodebuild -scheme Equinozio -destination 'generic/platform=iOS Simulator' -configuration Debug build CODE_SIGNING_ALLOWED=NO 2>&1 | tail -4`
- Test: `xcodebuild test -scheme Equinozio -destination 'platform=iOS Simulator,name=iPhone 17' -only-testing:EquinozioTests CODE_SIGNING_ALLOWED=NO 2>&1 | tail -4`
- Geometria canonica nel Canvas: arrangiamento a rombo, rapporto offset/raggio ≈ 0.89. Per l'app si mantiene `raggio = lato·0.225` e si usa `distanzaCentri = lato·0.201` (rombo, con margine per le etichette).

## Task 1: Emblema `QuattroCerchi` → rombo, senza puntino

**Files:**
- Modify: `Equinozio/DesignSystem/Components/QuattroCerchi.swift`

- [ ] **Step 1: Cambia proporzioni e disposizione a rombo**

Sostituisci le due righe delle proporzioni:
```swift
            let distanzaCentri: CGFloat = lato * 0.127
            let raggio: CGFloat = lato * 0.225
```
con:
```swift
            // Logo a rombo: stessi raggi, centri ai 4 punti cardinali.
            let distanzaCentri: CGFloat = lato * 0.201
            let raggio: CGFloat = lato * 0.225
```

Sostituisci l'array `posizioni` (quadrato) con la disposizione a rombo:
```swift
                    let posizioni: [(CGPoint, Color)] = [
                        (CGPoint(x: centro.x,                    y: centro.y - distanzaCentri), .passione),
                        (CGPoint(x: centro.x - distanzaCentri,   y: centro.y),                  .talento),
                        (CGPoint(x: centro.x + distanzaCentri,   y: centro.y),                  .missione),
                        (CGPoint(x: centro.x,                    y: centro.y + distanzaCentri), .professione),
                    ]
```

- [ ] **Step 2: Rimuovi il puntino centrale**

Elimina il blocco:
```swift
                // Punto centrale · l'equinozio
                Circle()
                    .fill(Color.salvia)
                    .frame(width: 6, height: 6)
                    .position(centro)
```

- [ ] **Step 3: Riposiziona le etichette ai vertici del rombo**

Sostituisci il corpo di `etichetteEsterne(...)` con il posizionamento cardinale:
```swift
    private func etichetteEsterne(centro: CGPoint, distanzaCentri: CGFloat, raggio: CGFloat) -> some View {
        let margine: CGFloat = 6

        return Group {
            etichetta(.passione)
                .position(x: centro.x, y: centro.y - distanzaCentri - raggio - margine)
            etichetta(.talento)
                .position(x: centro.x - distanzaCentri - raggio - margine, y: centro.y)
            etichetta(.missione)
                .position(x: centro.x + distanzaCentri + raggio + margine, y: centro.y)
            etichetta(.professione)
                .position(x: centro.x, y: centro.y + distanzaCentri + raggio + margine)
        }
    }
```
(Se `etichetteEsterne` non aveva `return`/`Group` perché `@ViewBuilder`, mantieni `@ViewBuilder` e rimuovi `return`/`Group`: lascia le quattro `etichetta(...).position(...)` in sequenza come prima. L'importante sono le nuove coordinate cardinali.)

Nel commento in cima al file, sostituisci "Diagramma a quattro cerchi (Venn) FISSO" con "Diagramma a quattro cerchi a rombo (logo canonico)".

## Task 2: Rigenerare l'icona app

**Files:**
- Create (temporaneo): `/tmp/eqz-logo.svg`
- Modify: `Equinozio/Assets.xcassets/AppIcon.appiconset/AppIcon.png`

- [ ] **Step 1: Scrivi l'SVG sorgente del rombo**

Crea `/tmp/eqz-logo.svg`:
```svg
<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <g style="isolation:isolate">
    <circle cx="50" cy="26.4" r="26.4" fill="#d38e8c" style="mix-blend-mode:multiply"/>
    <circle cx="26.4" cy="50" r="26.4" fill="#c2be7e" style="mix-blend-mode:multiply"/>
    <circle cx="73.6" cy="50" r="26.4" fill="#88bc97" style="mix-blend-mode:multiply"/>
    <circle cx="50" cy="73.6" r="26.4" fill="#82afc6" style="mix-blend-mode:multiply"/>
  </g>
</svg>
```

- [ ] **Step 2: Renderizza il mark e componi l'icona opaca**

```bash
cd /Users/giuseppe/XCode/Equinozio
rsvg-convert -w 656 -h 656 /tmp/eqz-logo.svg -o /tmp/eqz-mark.png
magick -size 1024x1024 gradient:'#e9efe3-#f2efe7' \
  /tmp/eqz-mark.png -gravity center -geometry +0-10 -composite \
  -alpha remove -alpha off \
  Equinozio/Assets.xcassets/AppIcon.appiconset/AppIcon.png
```
(656 ≈ 64% di 1024: il mark resta centrato con margine; `-alpha remove -alpha off` garantisce icona opaca, requisito App Store.)

- [ ] **Step 3: Verifica l'asset**

```bash
sips -g pixelWidth -g pixelHeight -g hasAlpha Equinozio/Assets.xcassets/AppIcon.appiconset/AppIcon.png
```
Atteso: `pixelWidth: 1024`, `pixelHeight: 1024`, `hasAlpha: no`.

- [ ] **Step 4: Build (l'asset catalog si compila)**

Build → `** BUILD SUCCEEDED **`.

- [ ] **Step 5: Commit**

```bash
git add Equinozio/Assets.xcassets/AppIcon.appiconset/AppIcon.png
git commit -m "design: icona app col logo a rombo su sfondo salvia"
```
